Here's a broad list, grouped by category. You won't need most of these for Div. 3 — I'll flag which tier matters when.

**Basic Data Structures**
Arrays, Strings, Linked Lists, Stacks, Queues, Deque, Hash Map, Hash Set, 2D Arrays/Matrices

**Sorting**
Bubble Sort, Selection Sort, Insertion Sort, Merge Sort, Quick Sort, Counting Sort, Radix Sort, Bucket Sort

**Searching & Core Techniques**
Linear Search, Binary Search, Binary Search on Answer, Ternary Search, Two Pointers, Sliding Window, Meet in the Middle, Prefix Sums, Difference Arrays, 2D Prefix Sums, Greedy, Recursion, Backtracking, Divide and Conquer, Bit Manipulation, Bitmasking, Brute Force / Complete Search

**Math**
Euclidean Algorithm (GCD/LCM), Extended Euclidean Algorithm, Modular Arithmetic, Fast Exponentiation, Modular Inverse, Sieve of Eratosthenes, Prime Factorization, Fermat's Little Theorem, Combinatorics (nCr / Pascal's Triangle), Chinese Remainder Theorem, Matrix Exponentiation, Euler's Totient Function, Fast Fourier Transform (FFT)

**Dynamic Programming**
1D DP, 2D DP, 0/1 Knapsack, Unbounded Knapsack, Coin Change, Longest Common Subsequence (LCS), Longest Increasing Subsequence (LIS), Edit Distance, Matrix Chain Multiplication, Bitmask DP, Digit DP, Tree DP, DP on Intervals

**Graphs**
Adjacency List/Matrix, BFS, DFS, Dijkstra's Algorithm, Bellman-Ford, Floyd-Warshall, Topological Sort, Union-Find / DSU, Kruskal's Algorithm, Prim's Algorithm, Tarjan's Algorithm (SCC), Kosaraju's Algorithm, Articulation Points & Bridges, Bipartite Check, 0-1 BFS, Multi-source BFS, Lowest Common Ancestor (LCA), Euler Tour, Max Flow (Ford-Fulkerson / Edmonds-Karp), Dinic's Algorithm, Bipartite Matching (Hopcroft-Karp), A* Search

**Trees**
Binary Trees, Binary Search Trees, Tree Traversals (in/pre/post-order), Segment Tree, Lazy Propagation, Fenwick Tree (BIT), Sparse Table, Trie, Heavy-Light Decomposition, Merge Sort Tree, Persistent Segment Tree

**Strings**
String Hashing, KMP Algorithm, Z-Function, Rabin-Karp, Manacher's Algorithm, Suffix Array, Suffix Automaton, Aho-Corasick Algorithm

**Heaps & Monotonic Structures**
Binary Heap / Priority Queue, Monotonic Stack, Monotonic Queue

**Other Useful Topics**
Game Theory (Nim, Sprague-Grundy), Convex Hull, Line Sweep, Coordinate Compression, Square Root Decomposition, Mo's Algorithm, Disjoint Sparse Table, Bitset Optimization
