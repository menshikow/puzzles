# 66. Plus One

**Problem type / Pattern:** Array manipulation, carry-over, in-place modification

**Difficulty:** Easy

**Key idea / Algorithm:**
- Start from the last digit
- Add 1, propagate carry if needed
- Handle edge case: all 9’s → prepend 1
